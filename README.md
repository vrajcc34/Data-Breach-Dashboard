# Data-Breach-Dashboard

## Overview
This project presents a Tableau dashboard analyzing large-scale data breaches across organizations, locations, and breach types. The goal was to identify patterns in breach activity and highlight where the largest risks exist.

## Dataset
The dataset includes historical data breach records with:
- Organization names
- Number of records breached
- Breach type
- Geographic location
- Timeline of breaches

## Dashboard Features
- Geographic heat map showing breached records by state
- KPI summary of total breached records
- Top organizations by number of records breached
- Breakdown of breach types
- Timeline of breach activity

## Key Insights
- A small number of organizations account for a large portion of total breached records
- Breaches are heavily concentrated in specific regions
- Certain breach types (such as hacking) dominate overall incidents
